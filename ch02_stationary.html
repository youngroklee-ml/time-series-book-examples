2.1.3 Ex 2.2: Autocorrelation function

Use n - 1 as denominator for covariance.

rho <- map_dbl(
  1:10,
  \(k) sum((earthquake$numbers[(k+1):n] - z_bar) * 
           (earthquake$numbers[1:(n-k)] - z_bar)) / 
           (n - 1) / gamma_0
)
rho

{stats}, one of R’s core package, provides acf() function to compute autocorrelation.

acf_results <- acf(earthquake$numbers, type = "correlation", plot = FALSE, lag.max = 10)

2.1.4 Ex 2.4: Partial autocorrelation function

x <- c(acf_results$acf[10:2], acf_results$acf[1:10])
y <- acf_results$acf[2:11]

mat <- reduce(
  2:10,
  \(m, i) rbind(m, x[(10 - i) + (1:10)]),
  .init = x[10:19]
)

map_dbl(
  1:10,
  \(k) solve(mat[1:k, 1:k], y[1:k])[k]
)
 [1]  0.395037778  0.184088155  0.113167641  0.060683539 -0.004447767
